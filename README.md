# CRUD CSharp

Tecnologias usadas
- Microsoft .NET Core 6
- EntityFramework y base de datos local
- Arquitectura en capas

## Montar la base de datos

En consola de administrados de paquetes \
Herramientas -> Administrador de paquetes Nuget -> Consola

```bash
Update-Database
```
